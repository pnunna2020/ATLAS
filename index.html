FAA AVS Portfolio — ATLAS

Application Transformation & Legacy Architecture Study

Research Artifact · Not for Official Use

ATLAS is a structured analysis of the FAA Aviation Safety division's major legacy IT systems — covering current-state architecture, data models, user workflows, and a future-state rationalization vision. Each mockup below is an interactive prototype built to explore UX, data flows, and integration opportunities.

Current-State System Mockups

Airman Certification · Current State

IACRA

Integrated Airman Certification and Rating Application — manages airman certificate applications, examiner workflows, and FAA registry submissions across all certificate types.

View Mockup →
Medical Certification · Current State

MedXPress / AMCS

Airman Medical Certification System — online medical application portal (MedXPress) paired with the Aviation Medical Examiner decision-support and FAA adjudication backend (AMCS/MSS).

View Mockup →
Designee Management · Current State

DMS

Designee Management System — tracks FAA-appointed designees (DPEs, DAMEs, DAHs, ODA holders) through selection, appointment, activity monitoring, and termination workflows.

View Mockup →
Aircraft Registration · Current State

RMS / CAIS

Registry Management System and Civil Aviation Information System — handles N-number assignment, aircraft registration applications, ownership transfers, renewals, and deregistration.

View Mockup →

Future-State Vision

Rationalization · Future State

Unified Rationalized Portal

A consolidated FAA AVS portal concept that merges IACRA, MedXPress, DMS, and RMS into a single authenticated experience — shared identity, unified case management, and a common data layer.
